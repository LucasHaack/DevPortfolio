Rainfall

Recent Weather

Rainfall was developed out of necessity. We wanted to know if we could mountain bike or if the trails would simply be too muddy. It is suprisely hard to find accurate historical weather data on the internet and we wanted to fix that. Our main goal with Rainfall has become to give the user the information they want to know without all of the other clutter.

Head over to rainfall.app to check it out!

Rainfall started as a simple, single page website that that told the user if and how much it had rained in their area of choice. It has become a full fledged weather site with much more to come!

Rainfall's front end is built using only vanilla Javascript, CSS, and HTML. We wanted to keep the code base as simple as possible. Vanilla JS is very fast compared to some of the other options and that has allowed us to keep the load times very low. CSS Grid is very easy to use and makes styling a breeze, especially when we want to add a new feature. By choosing not to use a heavy framework, we are able to keep Rainfall fast with very simple to read code.
